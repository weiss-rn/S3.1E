# P4 - Requests + BeautifulSoup

Practice fetching pages, selecting elements, and preparing output directories.

## Files

- `P4.py`: minimal request and parsing demo.
- `P4_1.py`: helper class for directory creation.
- `P4_2.py`: element selection using class filters.
- `P4_3.py`: page scrape with a limit and debug prints.

## Run locally

1) Install deps: `pip install requests beautifulsoup4`.
2) Execute the script you want, e.g., `python P4_3.py`.

## Output

- Creates the target directory (default `test` in the scripts).
- Prints found elements or article IDs to the console.
