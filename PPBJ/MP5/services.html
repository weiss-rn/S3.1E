FAQ

Quick answers about this static setup.

Does the form actually send messages? +

No—this is a static demo. The form shows an on-page confirmation. For real submissions you’d connect a backend or a form service.

Will this work on your Python socket server? +

Yes. These pages are plain HTML and reference only CDN assets.

Can I open the files without a server? +

Yes. Links use relative paths (like about.html) so local file browsing works too.
