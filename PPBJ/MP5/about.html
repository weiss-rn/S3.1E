About

Built to look great, kept intentionally simple.

AstraWeb is a small demo site designed for static hosting and learning. The goal: deliver a modern UI without a complicated setup.

Focus

UI

Static

Assets

Story

The original pages in this folder were template-driven. This redesign turns them into a cohesive, high‑quality static site with modern typography, layout, and interactions.

Design system

A shared header/footer, spacing, and component style across pages.

Small interactions

Theme toggle, mobile menu, and smooth anchors—no heavy scripts.
